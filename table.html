Instructions

This tool extracts and converts word tables into clean HTML code.

  1. Copy part or all of a word document containing table(s) to the clipboard.
  2. Paste it into the white box and select the settings you want. For example, the 'Add strong' setting will replace bold text from the word document with <strong> tags. If the cells seem very misaligned, disabling 'Add colspan and rowspan' can sometimes help.
  3. Click 'Generate HTML'. The tool will find and extract all tables from the document and output them separately below.
  4. Further customisation can be made to each table individually with the tools below it. Classes can be applied to rows and columns, which are visually represented on the table using colours. If a cell has multiple classes, stripes or a gradient will be used, dependant on user preference. More classes can be added using the '+' button.
  5. Once happy with the table markup, press 'Select for copying' and copy the html code for use in your project.

A selection of tables to test with can be found here: G:\WebDevelopment\people\jenny\entities\sample_tables

New: HTML Mode
If you check 'HTML Mode' the program will accept HTML code instead of a word document as input. You can then use the class tools to add classes to the HTML table.
Please note: All existing classes in the pasted in HTML code will be removed.
